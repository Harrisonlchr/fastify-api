## Description

Fastify API

## Requirements

- Node v16

## Installation

```bash
$ npm install
```

## Running the app

- First we need to create .env file for the project

     JWT_SECRET=(keyword to use as secret for the JWT)
     JWT_EXPIRATION_TIME=(default expiration time in seconds)

  3. Run the project, there are some options:

   ```bash
      # development
      $ npm run dev

      # run test
      $ npm run test
   ```
